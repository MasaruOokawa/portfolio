レンジタイムアシスト

レンジタイムアシスト
GitHub

**使用言語**
HTML / CSS（SASS） / JavaScript
**概要**
- お弁当や冷凍食品を効率的に温めるための計算ツールです。
**コンセプト・目的**
- 複数のお弁当を1回のレンジで効率よく温めることができます。
- 限られた休み時間をいかに充実させるか考えた時に思いつきました。
- お弁当を美味しく、早く食べれるようにと考えています。
- テキスト入力ではなく、プルダウン方式で素早く計算かできるようにしました。
- 簡単に電子レンジで効率よく計算できるようにしました。
